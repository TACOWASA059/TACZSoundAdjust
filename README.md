# TACZ Sound Adjust
このMODにより、TACZの銃声やサイレンサーの音が存在しない場合、
代替音を自動的に設定します。

クライアントmod

## 前提mod
```
forge 1.20.1
tacz-1.20.1-1.1.4-hotfix-all.jar
```
